# **Installaction Gideline**

### Stape 01 :

` Clone Git Repository :`

```
git clone https://github.com/codewithashim/Star-Wars-Clients.git
```

### Stape 02 :

` Install Dependancy :`

`cd/client`

```
npm install
```

### Stape 03 :

`Start The Project :`

```
npm run dev
```

# **Project Server Installaction**

`Clone Git Repository :`

```
git clone https://github.com/codewithashim/Star-Wars-Server.git
```

### Install Dependancy

`cd/server`

```
npm install
```

### Start The Server

```
nodemon server
```
