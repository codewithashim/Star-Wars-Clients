# **Installaction Gideline**

### Stape 01 :

` Clone Git Repository :`

```
git clone https://github.com/codewithashim/Star-Wars-Clients.git
```

### Stape 02 :

` Install Dependancy :`

`cd/client`

```
npm install
```

### Stape 03 :

`Start The Project :`

```
npm run dev
```

`Clone Git Repository :`

```

```

`cd/server`

```

```

### Start The Server

